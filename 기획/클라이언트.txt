클라이언트 버전 socket 파일에 전역 변수로 사용 static

서버 에서 필요한 값 
메인 메뉴 
{
    게임 시작 누르면 서버와 연결 connect 메세지 보냄 
}


랭킹
{
    랭킹 자신의 uuid 보내주고 체크 
}

인게임 
{
    자신의 uuid 보내주고 현재 클리어한 스테이지 최대 체크.

    몬스터 스폰때마다 타임스템프 비교해서 체크 해주기 서버랑. (오류 세번이상 나오면 메인 메뉴로 보내기 다 멈추고 )

    공격시 현재 공격 타이밍 위치랑 들어온 몬스터 위치 보내주기 (여러개면 어떡하지?) 받는정보는 fail success 둘중하나 string 값으로 data에  0 : no 100 : good 200 : great 400 : 엑설런트

    끝나고 현재 클리어 데이터 스코어 저장 
}
